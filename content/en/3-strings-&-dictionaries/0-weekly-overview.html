Weekly overview

It's your third week of learning Python! We're going to be covering some of my personal favorite topics this week.

Our learning goals for this week are:

  • Strings, string formatting, and string operations

  • The dictionary data type

  • Generating randomness programmatically

  • Working with files (both local and online)

At the end, you'll use those skills to build a procedural text generator that can output novel sentences from a book, movie titles, wise sayings, or whatever you decide. (I have literally built text generators in every language I know, because it's just so fun to see what comes out).
