İşlev, bir görevi gerçekleştiren ve kolayca yeniden kullanılabilen bir kod dizisidir. ♻️

Çağrı ifadelerinde aşağıdaki gibi işlevleri zaten kullandık:

abs(-100)

pow(2, 10)

Bir işlev bir girişi ( bağımsız değişkenleri) alır ve bir çıktı ( dönüş değeri) döndürür.

-100 → abs → 100

abs İfadede, -100 ve 100 dönüş değeri olan tek bir bağımsız değişken vardır.

2, 10 → pow → 1024

pow İfadede, 2 ve 10 olmak üzere iki bağımsız değişken ve 1024 dönüş değeri vardır.

Ancak bunların hepsi Python tarafından sağlanan işlevlerdir. Çoğu zaman, programımıza özgü bazı görevleri yerine getirmek için kendi işlevlerimizi oluşturmak istiyoruz.

İşlevleri tanımlama

Python'da fonksiyonları tanımlamanın en yaygın yolu bir def deyimdir.

Genel şablon aşağıda verilmiştir:

():
    return

İşte gerçek bir örnek:

def add_two(num1, num2):
    return num1 + num2

Tanımlandıktan sonra, tıpkı yerleşik Python işlevleri dediğimiz gibi işlevi de çağırabiliriz:

add_two(2, 2)
add_two(18, 69)

Bu işlev tanımı ifadesini biraz parçalayalım. Adı ve parametreleri bildiren ilk satıra işlev imzası denir ve altındaki tüm girintili çizgiler işlev gövdesidir.

():      # ← Function signature
    return  # ← Function body
def add_two(num1, num2):       # ← Function signature
    return num1 + num2         # ← Function body

Python dili, kod satırlarının birbirleriyle nasıl ilişkili olduğunu göstermek için girinti kullanması bakımından biraz benzersizdir. (Diğer diller parantez veya anahtar kelimeler kullanır). Bu, işlev gövdesindeki çizgilerin girintili olması gerektiği anlamına gelir. Aslında ne kadar girintili oldukları önemli değil - heck, Python onları 200 boşlukla girintilendirmekten kurtulmanıza izin verecek! Bununla birlikte, standart kural 2 boşluk veya 4 boşluktur; Kodu okuyan biri tarafından görülebilecek kadar girinti, ancak kod satırlarının gülünç derecede uzun olması kadar değil.

İşlev gövdesi birden fazla satıra sahip olabilir (ve genellikle yapar):

def add_two(num1, num2):      # ← Function signature
    result = num1 + num2      # ← Function body
    return result             # ← Function body

Parametreler ve bağımsız değişkenler

Artık bir işleve geçirilen değerlere başvurmak için iki farklı terim kullandık: parametreler ve bağımsız değişkenler. Peki, hangisi?? Her ikisi!

Bir işlevi tanımlarken, parametreler sonunda bir işleve geçirilecek değerlere verilen adlardır. Aşağıdaki işlev iki parametreyi kabul eder ve num2, ve dönüş değeri bu iki parametreyi num1 toplar.

def add_two(num1, num2):

Ancak, aslında bu işlevi çağırdığımızda, bağımsız değişkenler olarak iletilen değerleri tanımlarız. Aşağıda, 18 ve 69 olmak üzere iki argümandan geçiyoruz.

add_two(18, 69)

Aradaki fark ince ve aşırı bilgiççe görünebilir ve ortaya çıktığı gibi, programcılar bazen bilgiççe olabilirler.

Dönüş değerleri

Anahtar return kelime, işlevi çağıran kişiye bir değer döndürür. Bir return deyimi yürütüldükten sonra, Python bu işlevden hemen çıkar ve işlevde başka kod yürütmez.

def add_two(num1, num2):
    return num1 + num2

result = add_two(2, 4)

Bu kodu çalıştırdıktan sonra, değişken result dönüş değeri olan 6'yı tutar.

Yerleşik işlevlerde yaptığımız gibi, işlev çağrılarını aritmetik ifadelerde de kullanabiliriz:

big_sum = add_two(200, 412) + add_two(312, 256)

Ayrıca çağrıları diğer çağrıların içine yerleştirin:

huge_sum = add(add(200, 412), add(312, 256))
